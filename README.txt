What this version includes:
- Modern desktop homepage layout
- Mobile-friendly responsive layout
- Service Type / Make / ZIP search
- Sample dealership special cards
- About section
- Google Sheet ID noted in script.js
- Mileage-ready card design

Important:
To pull live data from Google Sheets, publish your sheet as CSV and update SHEET_CSV_URL in script.js.
For real mileage, your sheet should include dealer latitude and longitude columns, or the website needs a geocoding API.
